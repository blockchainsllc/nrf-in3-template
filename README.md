### NRF52840 with IN3 Template project

Use this is as your boiler plate for IN3 projects on nrf52840.

#### Supported transport modules:

* UART(*default*)

**UART:**

| PIN NUMBER | PIN NAME |
|:----------:|:--------:|
|   P0.29    |   TXD    |
|   P0.31    |   RXD    |
|   P1.13    |   RTS    |
|   P1.15    |   CTS    |

*Note: UART interface must connect TXD->RXD, RXD->TXD, RTS->CTS, CTS->RTS/DTR*
